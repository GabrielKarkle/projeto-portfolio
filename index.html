Projeto Cordel

pequeno projeto feito para entender como funciona uma foto e tambem o efeito paralax em um site

saiba mais...

Este é um pequeno projeto desenvolvido com o objetivo de compreender melhor como funciona a exibição de uma imagem em um site, bem como explorar e experimentar o efeito parallax — uma técnica visual que cria a sensação de profundidade e movimento ao rolar a página. Através desse exercício, foi possível aplicar conceitos básicos de HTML, CSS e JavaScript, entendendo como camadas de conteúdo podem se mover em velocidades diferentes para proporcionar uma experiência visual mais dinâmica e imersiva.

Projeto Links

foi criado este projeto com o intuito de conhecer os funcionamento dos links em geral

saiba mais...

Este projeto foi criado com o intuito de entender o funcionamento dos links em geral dentro de uma página web. O foco principal foi aprender como utilizar a tag para navegar entre diferentes seções da própria página (links âncora), redirecionar para outras páginas internas ou externas, além de explorar atributos importantes como href, target, rel e outros. Através dessa prática, foi possível compreender como os links contribuem para a navegação e a usabilidade de um site.

Projeto Login

projeto feito para aprender o uso dos formularios para o login como o nome email e contrasenha do usuário

saiba mais...

Este projeto foi desenvolvido com o objetivo de aprender a utilizar formulários em páginas web, focando especialmente em situações de login. Durante a construção do projeto, foram aplicados campos básicos como nome, e-mail e senha do usuário, utilizando as tags form, input, label e atributos como type, placeholder, required, entre outros. A proposta foi entender como os dados são coletados de forma estruturada e como preparar o formulário para interações futuras com sistemas de autenticação.
